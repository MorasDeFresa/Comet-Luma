Este es un proyecto basado en expo go de la mano de la libreria react native

## Guia de uso

1. Clona el repositorio

   ```bash
   git clone https://github.com/MorasDeFresa/Comet-Luma.git
   ```

2. Abre el terminal en la ruta donde clonaste el repositorio

1. Instala las dependencias

   ```bash
   npm install
   ```

2. Inicia la app

   ```bash
   npm run start
   ```

Una vez ya cargue la build en tu terminal aparecera los respectivos enlaces para abrir el proyecto por navegador web o si tienes la app expo go puedes scanear el QR y ver el funcionamiento de la app en tu celular (sea IOS o Android)

Para acceder a la apk compilada. Solo es ingresar al siguiente enlace
